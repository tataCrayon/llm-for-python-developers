# 技术栈

- 核心框架 
  - FastAPI: 用于构建API接口。
  - LangChain: 用于构建RAG系统。

  
- 向量数据库
  - Chroma


- 嵌入模型 
  - Qwen3-Embedding-4B
  - BGE-Large-ZH


- 大语言模型 
  - DeepSeek
